Enum rocket::http::uri::Query

source · [−]
pub enum Query {}
Expand description

Marker type indicating use of a type for the query UriPart of a URI.

In route URIs, this corresponds to all of the text after a ?, if any.

#[get("/home/<name>/<page>?<item>&<form..>")]
                           ^-------------- Query
